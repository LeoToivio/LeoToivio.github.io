Acron: Attack of the Squirrels!

A VR vs mobile party game for 3-9 people!

Platform(s) Quest, Rift, Steam, iOS, & Android
Main role(s) Game Developer
Team size 6 - 12
Game Engine Unity

I worked as a developer on Acron. It consisted of making the game fun, mechanics tight, some UX work and a lot of network programming. Using the magic of Unity and Perforce we were able to make a game that is a VR and traditional smartphone game simultaneously.
