Consistency

Basic Guidelines

This is essential to work in groups

  • Be as consistent as possible
  • Use only english alphabet - no special characters or swedish letters (å, ä or ö)
  • No spaces in file names or variable names
  • Use Camel Case

Naming convention: https://wiki.unrealengine.com/Assets_Naming_Convention

Assignment:

  • Create all the folders described in the wiki
  • Make sure that all file names are according to the standard
  • Put the files in the correct folders
  • Make sure all variable names in your blueprints are correct
